What started as one developer's hobby projects has grown into a full-stack ecosystem.

922-Studio is not a company. It's a brand for a group of hobby developers who build real things — web apps, APIs, games, infrastructure — and ship them to production.

## How it started

It began with simple websites and online shops. Then came a personal portfolio. Then a home server. Then microservices. Then monitoring. Then CI/CD. Then a Discord bot with an idle game.

Before we knew it, there were 12 active projects running on self-hosted infrastructure with zero-downtime deployments, AI-powered versioning, and comprehensive test coverage.

## What we build

Everything runs on a self-hosted home lab server — no cloud dependency, no vendor lock-in. The stack spans:

- **5 FastAPI microservices** handling finance, health, monitoring, social media, and authentication
- **React frontends** with TypeScript, TanStack Query, and Tailwind CSS
- **A real-time multiplayer game** built with Socket.io
- **A Discord bot** with idle game mechanics
- **Centralized CI/CD** with reusable GitHub Actions and AI-powered semantic versioning

## Who we are

We're a small group of hobby developers. Gregor handles the infrastructure, backend, and most of the frontend work. Iustus is building ANIZO, an anime tracking app. We're always looking for more people who want to build cool things.

## What's next

This studio page is our first step toward documenting and showcasing everything we do. Expect blog posts about technical decisions, project launches, and lessons learned.

Welcome to 922-Studio.
